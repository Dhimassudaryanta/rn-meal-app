How to run this project :

1. Clone this repo via terminal / download zip file

``` 
$ git@github.com:Dhimassudaryanta/rn-meal-app.git

$ cd rn-meal-app

$ npm install
```

2. To start the app, you can run:

```
$ npm start
```
